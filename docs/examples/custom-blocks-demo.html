Custom Markdown Blocks Demo

What are Custom Blocks?

Custom blocks are special code fence blocks in markdown that render as interactive components. They use the syntax:

```blocktype
key: value
another_key: another_value
```

The Codex system recognizes these blocks and renders them using Vue.js components.

Implementation Details

Backend (Python)

  • CustomBlockParser - Parses markdown and extracts custom blocks
  • Distinguishes custom blocks from standard code blocks (python, javascript, etc.)
  • Parses YAML configuration inside blocks
  • API endpoint /api/v1/markdown/render returns custom block metadata

Frontend (Vue.js)

  • WeatherBlock.vue - Component for rendering weather blocks
  • LinkPreviewBlock.vue - Component for rendering link preview blocks
  • MarkdownViewer.vue - Main component that detects and renders custom blocks
  • Blocks are registered in a component registry
  • Vue components are dynamically mounted for each custom block

Integration Plugins

Custom blocks are defined in integration plugin manifests (YAML files):

# backend/plugins/integrations/weather-api/integration.yaml
